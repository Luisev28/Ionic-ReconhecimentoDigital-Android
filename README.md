# Ionic-ReconhecimentoDigital-Android
Reconhecimento da Digital 

Não esqueça o *npm install*

<br>
<p>Comandos:</p>
<p>ionic build</p>
<p>ionic cap add android</p>
<p>ionic cap sync </p>
<p>ionic cap open android</p>

<br>
Deverá ter o Android Studio e ambiente preparado.
<br>
OBS: Funcionará no celular, o simulador causará erro no console por não ter digital cadastrada.
